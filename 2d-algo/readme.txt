Readme
=========================
This package contains all source codes for 
2DRAH
2DRKH

Usage Step
==========
a. Compilation
2DRAH	g++ -std=c++11 -O2 -g 2drah.cpp
2DRKH	g++ -std=c++11 -O2 -g 2drkh.cpp
b. Execution
	./a.out data n r alpha
 * data : data name, e.g., "anti2d", "Island";
 * n : number of tuple
 * r : output size
 * alpha : threshold
c. Output
	The output will be shown on the console.
